Library

Check out items

Borrow or renew items for three weeks, and view your checkouts.

You must get a library card to check out items. Fill out the nearby paper form. New cards should be available within a week. Your card will be stored in the filing box.

Find items

Browse and search for items in the public catalog.

Return items

Place items in the slot of the nearby Book Return box. Do not return items to the shelves. Library helpers will check in the items and return them to the shelves.

Help

  • Ask a library helper on Sundays (look for a person wearing a badge).
  • Write a note and leave it at the library.
  • Email the library at: library@cccbloomington.org
  • Learn more about the library on the church website.
